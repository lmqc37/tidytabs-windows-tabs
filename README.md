## Windows 窗口标签化：为什么你需要这个功能

同时开着三个 Word 文档、两个 Excel 表格、几个文件夹窗口——这是普通上班族的日常。任务栏一旦满了，切换就变成了噩梦。

Windows 的 Alt+Tab 不是不能用，但当窗口超过十个，这个快捷键反而更乱。分屏功能有用，但它解决的是窗口排布问题，不是数量管理问题。

**窗口标签化**才是真正对症的方案。把相关的窗口归并到同一个标签组里，切换只需点一下标签。桌面从混乱变成清爽，找窗口的时间砍掉大半。

TidyTabs 在这个场景下做得干净——它不改变你的使用习惯，只是给每个窗口加了一个可以操控的标签。装好就能用，不需要学习新的操作逻辑。

## TidyTabs 实际体验：六个核心功能拆解

### 1. 拖拽合并，两秒完成

把鼠标移到任何窗口的左上角，标签会悄悄浮现。抓住这个标签，拖到另一个窗口的标签上——两个窗口合并成一个，顶部出现标签栏。再拖一个进来，再多一个标签。

不同软件也可以合并。Word 文档和 Excel 可以在同一个标签组里并存，文件夹和 PDF 阅读器也行。

### 2. 自动隐藏，不打扰工作

单独的窗口（没有被分组的那种）标签默认隐藏，只有鼠标靠近才显示。已经分组的标签栏始终可见，但在非活跃状态下会保持半透明。

这个设计特别聪明。你不会觉得桌面被一堆标签塞满，它在你需要的时候才出现。

### 3. 黑白名单，精细控制

不是所有软件都适合加标签。TidyTabs 会自动排除一些特殊窗口，你也可以手动设置黑名单（不加标签）或白名单（只给这几个软件加标签）。右键点标签，两下就设好了。

### 4. 专业版独有：多窗口分组与重排

免费版每个标签组最多 3 个窗口。专业版没有这个限制，还支持标签重排序、标签重命名、中键关闭标签，以及多显示器支持。

一旦工作文件超过三个，免费版就不够用了。这是升级的核心理由。

### 5. 与 AquaSnap 联动

AquaSnap 是同一家公司（Nurgo Software）出的分屏工具。两者配合使用，可以把一个标签组整体对齐、停靠或分屏——整个标签组作为一个单位参与分屏排布，非常顺手。

### 6. 轻量到几乎感觉不到它在运行

TidyTabs 的内存占用约 4MB，CPU 占用接近 0。在配置一般的办公机上跑一整天，感觉不到任何性能影响。
